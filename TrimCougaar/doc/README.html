Trim Cougaar Installation

Overview:

Trim Cougaar contains a subset of a full Cougaar install. It contains only the files needed to create a distributed agent runtime infrastructure and not extra files used by Logistics Applications or Ultralog robustness and security extensions.

Trim Cougaar has the same directory structure as the full Cougaar installation, so the top level directory can be used for COUGAAR_INSTALL_PATH and CIP.

Trim Cougaar is an eclipse project and is configured so that Cougaar applications projects can depend on the TrimCougaar project. The recommended eclipse workspace has the form:
MyWorkspace
  TrimCougaar
  hello (depends on TrimCougaar)
  MyNewCougaarAppl (depends on TrimCougaar)

The projects in the Cougaar Tutorials depend on having checked out Trim Cougaar in the same workspace.

Quick Start:

A Unix bash commands to:
  1. Create a workspace directory
  2. Checkout Trim Cougaar and Hello World projects from Cougaar CVS:
  3. Setup the Unix environment variables and path.
  4. Build and run the hello application using ant.
export WORK_SPACE=`pwd`/cougaar-ws
mkdir $WORK_SPACE
cd $WORK_SPACE
cvs -d :pserver:anonymous@cvs.cougaar.org:/cvsroot/tutorials co TrimCougaar
cvs -d :pserver:anonymous@cvs.cougaar.org:/cvsroot/tutorials co hello
export COUGAAR_INSTALL_PATH=$WORK_SPACE/TrimCougaar
export PATH=$COUGAAR_INSTALL_PATH/bin:$PATH
cd $WORK_SPACE/hello
export COUGAAR_SOCIETY_PATH=`pwd`
export COUGAAR_RUNTIME_PATH=$COUGAAR_SOCIETY_PATH/run
ant helloLifeCycle

To run the hello application using the Unix shell script:
cd $COUGAAR_RUNTIME_PATH
cougaar -v $COUGAAR_SOCIETY_PATH/configs/HelloLifeCycleSociety.xml SingleNodeRuntime.xml
